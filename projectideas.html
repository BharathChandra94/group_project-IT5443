Projects

Here are some project ideas to get you started. You can also use these ideas to help you think of your own projects.

Side Panel Sizing

This project will allow users to individually assign the widths of each side panel in the browser (Ex: bookmarks, reading list, history, customize chrome, and more)! Contributors will be using C++ and can use chromium code search to see how current side panels are resized. Once this project is completed users will be able to customize the widths of all their side panels to their preferences!

Project Length: 10 Weeks

Skills Required: C++

Linux audio capture local muting

We recently landed support for Chrome to capture system audio on Linux for use when casting or screen sharing. However, for use when casting, we need to mute local audio output while system audio is being captured. This project will 1. Update the audio capture implementation to support local muting 2. Improve unit test coverage for the pulseaudio backend in Chrome

Project Length: 12 Weeks

Skills Required: C++, Linux

Chrome Accept-Language Validation

Keep invalid languages off of Chrome's Accept-Language list. This can happen in a few ways and mostly affects Android users using multiple languages. Fixing this will close multiple long standing Chromium bugs and improve the language settings for Androind/Chrome users. We should also investigate not automatically adding the changed Android system language to Chrome's Accept-Languages. There are opportunities to extend this work into decoupling the Accept-Language list and Never Translate language lists helping make our language settings more streamlined.

Project Length: 12 Weeks

Skills Required: C++, Java

Integrate Emscripten with JS Promises

mscripten is a compiler and SDK for bringing C/C++ code to the Web via WebAssembly. The Web is an extremely asynchronous environment, with most Web APIs returning JS Promises. C and C++, on the other hand, are typically synchronous, with APIs blocking until they return results. Bridging this gap between synchronous C/C++ and the asynchronous Web is a common pain point for Emscripten's users.

Project Length: 12 Weeks

Skills Required:Ideally familiar with JS and C. Familiarity with C++ would be a bonus
